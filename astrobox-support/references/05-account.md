# 账号篇

> 来源：https://wiki.bandbbs.cn/Guides/astrobox/AstroboxV1账号篇.html

## Q：米坛账号无法登录？

解决方案：检查网络，并使用 Chrome/Edge 浏览器登录。若持续转圈，请尝试在米坛网页版上传或更换头像（此为 1.0.2 版本前的问题）。

> 补充：建议首选使用米坛社区官方客户端进行登录，其次将你的默认浏览器设置为 Chrome 或 Microsoft Edge，别用系统自带浏览器，夸克和 QQ 浏览器之类的也最好别用。

## Q：小米账号无法登录或在二次验证无限循环？

解决方案：可使用手机号、米 ID 或邮箱登录。若一直转圈，尝试检查账号下是否有穿戴设备、更改密码或切换 WiFi/流量。若依旧因此无法连接设备，请参考"如何获取 Authkey"。

> 回复时直接提供手动提取 Authkey 方法，安卓 / iOS 都要提供。

> 补充：如果外区无法使用手机号登录，请等待后续版本更新。如果依旧不行请带 log 与我们反馈。

### 关于小米账号登录

AstroBox 部分功能涉及通过调用小米账号服务接口以访问资源或启用特定设备操作。在此过程中，用户提供的账号凭证信息（包括用户名、密码或授权票据等）将直接提交至小米官方服务器，以完成身份验证。

本软件不会在未经用户同意的情况下存储、记录或篡改用户凭证信息。然而：

- 该传输行为的安全性完全依赖于小米服务器及其接口实现机制，非本软件所能控制或担保
- 若因接口变化、服务策略调整或安全机制触发而导致用户账号受到限制、登录失败或产生数据问题，本软件及其开发者不承担相关责任
- 用户应审慎判断是否使用此类功能，并对涉及账号凭证的使用风险承担全部责任

我们强烈建议用户仅在可信网络环境中操作，并在授权登录前评估其账号安全性要求。
