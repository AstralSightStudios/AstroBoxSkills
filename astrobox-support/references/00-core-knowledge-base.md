## 二、入门 / 安装篇

### Q：在哪下载 AstroBox 最新版本
在 AstroBox 官网：`astrobox.online`

### Q：手机安装的时候显示“解析软件包出现问题”？
解决方案：请检查你的安卓版本是否在安卓 10 及以上。13 以下版本稳定性差，推荐使用 13 及以上安卓版本。

### Q：Mac 打开显示“‘AstroBox.app’已损坏，无法打开。你应该将它移到废纸篓。”？
解决方案：在命令行执行以下指令并输入密码：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/AstroBox.app
```

### Q：Windows 安装的时候显示报错码 2502/2503？
解决方案：
1. 首先去官网重新下载安装包，看看安装包有没有在下载过程中损坏。
2. 如果不行，那就是 msi 的安装权限问题，请按照以下流程操作：
   - 右键安装包所在文件夹，点击属性。
   - 在“安全”选项卡下选择当前用户，点击“编辑”。
   - 勾选“完全控制”后点击确定。
   - 再次启动安装包尝试安装。

### Q：首页加载速度慢 / 加载失败？
解决方案：
1. 在设置中更换 CDN 并重启，建议换成 ghp；如果能接受一直开启代理使用，直接改成 raw 也可以。
2. 或者尝试修改 WiFi 的 DNS，将 DNS1 改为 114.114.114.114 或 223.5.5.5，DNS2 可改为 8.8.8.8。

### 🌟 Q：首页空白 / 设置页划不动？
特征：banner 能显示、应用列表只显示“已经被一查到底了”、设置页划不动。

解决方案：大概率是系统 Webview 版本过低，请将 Webview 更新到 115+。如果安装失败，请检查安卓版本或在开发者选项中关闭系统优化（如 MIUI 优化）。华为手机用户可尝试安装华为专用版 Webview。

### Q：动画卡顿 / 异常？
解决方案：手机端出现此问题属正常现象，需要骁龙 8G1 以上芯片才能流畅展示动画。

### Q：Windows 11 设备更新到 1.5.0 主页白屏 / 什么都点不动？
解决方案：此问题在 1.5.1 版本中已修复，请更新到最新版本。临时方案是使用 Windows 8 兼容模式运行程序。

### Q：鸿蒙用不了 / 装不了？
解决方案：1.2.0 版本后已支持安卓版本 ≥10 的鸿蒙。需要 AOSP 版本 ≥13。鸿蒙 Next 可尝试“卓易通”。大部分华为机型仍不兼容。

## 三、账号篇

### 🌟 Q：米坛账号无法登录？
解决方案：检查网络，并使用 Chrome/Edge 浏览器登录。若持续转圈，请尝试在米坛网页版上传或更换头像（此为 1.0.2 版本前的问题）。

### 🌟 Q：小米账号无法登录或在二次验证无限循环？
解决方案：可使用手机号、米 ID 或邮箱登录。若一直转圈，尝试检查账号下是否有穿戴设备、更改密码或切换 WiFi/流量。若依旧因此无法连接设备，请参考“如何获取 Authkey”。

> 回复时直接提供手动提取 Authkey 方法，安卓 / iOS 都要提供。

## 四、连接篇

### Q：如何判断是否连接设备成功？
设备页不显示电量 = 连接失败。

### Q：目前什么手环可以使用？
支持：小米手表 S3 及以后、小米手环 9 及以后、红米手表（rw）5 及以后。

不支持：小米手环 8 Pro。

### 🌟 Q：如何连接手环？
步骤：
1. 授予 AstroBox 蓝牙、附近设备和位置权限（需要开启精确位置）。
2. 登录小米账号或使用其他方式获取 Authkey。
3. 在手机上退出小米运动健康并确保完全关闭（或切换至另一穿戴设备）。
4. 手环进入“连接新设备”模式。
5. 在 AstroBox 中选择设备并连接。
6. Authkey 在登录小米账号的状态下会自动为你准备好，直接连接即可；未登录的话直接填入 Authkey 即可。
7. Windows 需要在系统蓝牙设置中点击允许配对（可能会在右下角弹出通知，点击进去配对即可）。

### Q：如何获得 Authkey？
方法一（推荐）：在 AstroBox 设置中登录小米账号一键同步。

方法二（安卓）：手动提取。通过小米运动健康 App 提取 log（“我的”->“关于”->连续点击橙色图标 10 次），使用 https://ikun-cxkpro.top/miwear-auth/ 解析日志。

方法三（iOS）：使用最新版小米运动健康 App 连接一次设备，打开系统自带“文件”App，浏览“我的 iPhone”，找到小米运动健康文件夹，按路径找到文件：

`\MHWCahe\你的小米账号id\VirtualDevice_registerList\manifest.sqlite`

使用 https://ikun-cxkpro.top/miwear-auth/ 解析日志，或直接用文本编辑器打开查找 `encrypt_key`（一定要先说通过网站解析）。

### 🌟 Q：为什么我连不上手环 / 几秒钟就断连了？
检查清单：
1. “小米运动健康”是否彻底关闭？
2. 其他可能抢占连接的 App（如 Notify For Xiaomi）是否已关闭或卸载？
3. 手环是否处于“连接新设备”模式？
4. AstroBox 是否拥有“附近的设备”、“蓝牙”、“位置（精确位置）”权限？
5. 是否尝试在系统蓝牙设置中“忽略”了该设备并重连（iOS 用户尤其需要）？
6. 电脑请检查是否有蓝牙模块（4.0+），检查有没有在电脑设置点击确认，手环点击确认。

### Q：手环链接后显示请在手机上点击确认，点击后没反应怎么办？
1. 检查是否授予 AstroBox 蓝牙和位置权限。
2. 去蓝牙设置里忽略设备。
3. 进小米运动健康重连设备。
4. 深度关闭、停止小米运动健康。
5. 手环进入连接新手机状态。
6. 进 AstroBox 设置，重新通过小米账号同步 Authkey。
7. 进入连接设备页重新连接。

### 🌟 Q：显示“密钥错误或后端异常”？
解决方案：通常是 Authkey 已过期。请重新打开“小米运动健康”同步一次，然后回到 AstroBox 重新登录小米账号以获取最新的 Authkey。

### Q：显示“连接失败…SPP…LOCATION permission”？
解决方案：请检查是否授予 AstroBox 蓝牙和位置权限。

## 五、下载安装资源篇

### Q：首页的 bandbbs 栏目为什么空的 / 显示“No BandBBS account”？
你没在设置里登录米坛账号。

### Q：Bandbbs 的资源点击下载后不会和官方源一样自动下载？
Bandbbs 的资源点击下载后需要选择下拉框里的文件再下载安装，请下载前自行确定资源适配的穿戴设备型号是否与你的设备型号一致。

### Q：资源显示 404 或“resource not found”
这个资源不存在，请联系资源作者解决。

### Q：出现“error sending request for url”？
解决方案：在设置中切换 CDN 并重启，或使用代理。

### Q：出现“No devices are connected”/“deadline has elapsed”怎么办
请确认手环是否已经连接，回到主页确定当前连接状态，建议多重启应用，重试几次。

### Q：出现“channel closed”？
解决方案：连接断开或 Authkey 错误。请尝试重新连接或重新同步 Authkey。

### Q：出现“Prepare not READY!”？
解决方案：手环空间或电量不足。重启手环，或尝试在资源队列中点击“编辑”按钮随机修改一个 ID。

对于 Redmi Watch 5 esim（rw5e），如果你在安装快应用，请你降级版本，系统限制了应用安装。

### Q：出现“Timeout waiting for protokey”怎么办
请重启 AstroBox 和手环并再次尝试。

### Q：出现“failed to open file”怎么办
1. 请检查自己的文件是否下载完整。
2. 检查是否授予 AstroBox 足够的文件读取权限。

### Q：出现“failed to get metadata of path”怎么办
你可以检查一下手环是否安装上了这个应用，大概率是 AstroBox 本身的 bug。

### Q：出现“url is not a valid path”怎么办
请检查你的文件格式是否正确，通常为 bin 或者是 rpk（快应用）。

### 🌟 Q：固件安装可以用吗？
风险极高：固件安装是危险操作，可能导致设备变砖，AstroBox 团队概不负责。不建议在安卓手机上操作。如要尝试，建议将发包间隔改为 10。

### Q：安装多个表盘只显示一个？
原因：表盘开发者使用了相同的 ID。

解决方案：关闭“自动安装”，在下载队列中点击“画笔”图标，随机生成一个新的 ID 后再发送到设备。

## 六、iOS 版特殊须知

### 安装
- 核心：AstroBox 永远无法上架 AppStore，必须通过电脑侧载（如 AltStore, SideStore）。
- 系统要求：最低 iOS 16.5，推荐 17.0+。

### 功能
- 绑定设备：每次从“小米运动健康”切换到 AstroBox 前，都必须先在系统蓝牙设置中忽略该设备。
- 资源安装速度：由于蓝牙底层限制，iOS 上的安装速度会比其他平台慢 3-5 倍，属正常现象。
- 安卓伪装模式：用于解决部分设备（如 Xiaomi Watch S4）连接 iOS 时快应用被隐藏的问题。开启此模式可让快应用重新显示，但连接会变得不稳定，无法安装资源。仅用于临时“解锁”快应用，日常使用时需关闭。

当用户多次提问二次验证相关问题时，直接给出最终解决办法：手动提取 Authkey（例如二次验证怎么过、循环很多次等）。

## 七、遇到未知问题怎么办

比较好的办法是提取应用运行 log 并发在群中。

Logs 提取方式：

电脑用户可以更新至 1.0.1 以后版本直接在设置里打开，也可以从以下地址中自行查找：

- 安卓：在原生文件 app 直接选择 astrobox（可以通过 MT 等工具搜索到原生的文件并打开，在左上角可以切换到 AstroBox 的选项卡，里面的 Logs 文件夹就是我们要的）。
- Windows：`C:\Users\你的用户名\AppData\Local\moe.astralsight.astrobox`
- Mac：`/Users（或译为用户）/你的用户名/Library（或译为资源库，需要快捷键 command+shift+.才会出现）/Logs/moe.astralsight.astrobox/`
- Linux：`/home/你的用户名/.local/share/moe.astralsight.astrobox/`
